# Security Policy

If you discover a security vulnerability in SxnnyUI, thank you — please report it privately so we can address it responsibly.

Reporting
- Preferred: send an email to `houjouzetaalpha@gmail.com` with a clear description, steps to reproduce, and any PoC code or attachments.
- If email is not available, open a private GitHub issue and mark it as `security` in the title.

Response
- We will acknowledge receipt within 72 hours and provide an expected timeline for a fix and disclosure.
- Critical fixes will be prioritized and a coordinated disclosure will be arranged if necessary.

Disclosure
- Do not publicly disclose the vulnerability until a fix is released or maintainers agree on a disclosure plan.

Replace the placeholder email addresses above with a real contact for your project before publishing.
