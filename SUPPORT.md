# Support

## Getting Help

**GitHub Issues**
For bugs or unexpected behavior, open an [issue](https://github.com/Sxnnyside-Project/SxnnyErgo/issues). Use the bug report template and include enough context to reproduce the problem.

---

## Before Asking

- Check the [README](README.md) — installation, usage, and architecture are documented there.
- Search existing [issues](https://github.com/Sxnnyside-Project/SxnnyErgo/issues) before opening a new one.

---

## What This Isn't

This is not a priority support channel. Response times are not guaranteed.
For security vulnerabilities, see [SECURITY.md](SECURITY.md).
